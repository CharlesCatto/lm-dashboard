## 🎯 Pourquoi ce projet ?

En tant que préparateur de commande chez Leroy Merlin, je voyais mes collègues et mon responsable utiliser un tableur Excel pour suivre la production. L'idée ? Rendre ces données plus **visuelles**, plus **intuitives** et accessibles sur mobile.

Ce projet est une **preuve de concept** pour montrer ce qu'on pourrait faire avec un vrai outil adapté au terrain. 

## ✨ Fonctionnalités actuelles

- ✅ Interface mobile-first, inspirée de l'univers Leroy Merlin
- ✅ Graphique interactif avec Chart.js
- ✅ Calcul automatique des stats : performance %, moyenne, max, min
- ✅ Sélecteur de période (jour/semaine/mois)
- ✅ 3 préparateurs fictifs pour la démo
- ✅ Démo en ligne accessible par QR code

## 🚀 Prochaines évolutions prévues

- 🔄 Parsing du vrai fichier Excel (copier-coller ou drag & drop)
- 🔄 Objectifs saisonniers ajustables (ex: 56 en hiver)
- 🔄 Comparaison entre préparateurs
- 🔄 Export PDF

## 📦 Installation en local

# Cloner le repo
git clone https://github.com/CharlesCatto/lm-dashboard.git

# Aller dans le dossier
cd lm-dashboard

# Installer les dépendances
npm install

# Lancer en local
npm run dev

Puis ouvrir http://localhost:5173
